## Dev Notes

### Design Tokens (de wireframes.md)
```css
/* Colors */
--accent: #3B82F6      /* Blue - primary actions */
--danger: #EF4444      /* Red - destructive actions */
--bg-primary: #FFFFFF
--bg-secondary: #F3F4F6
--text-primary: #111827
--text-secondary: #6B7280

/* Spacing & Sizing */
--radius-sm: 0.25rem
--radius-md: 0.5rem
--radius-lg: 0.75rem

/* Shadows */
--shadow-sm: 0 1px 2px rgba(0,0,0,0.05)
--shadow-md: 0 4px 6px rgba(0,0,0,0.1)
--shadow-lg: 0 10px 15px rgba(0,0,0,0.1)
```

### Component Patterns
```typescript
// Button example
interface ButtonProps {
  variant?: 'primary' | 'secondary' | 'danger' | 'ghost';
  size?: 'sm' | 'md' | 'lg';
  disabled?: boolean;
  children: React.ReactNode;
  onClick?: () => void;
}

// Modal example
interface ModalProps {
  isOpen: boolean;
  onClose: () => void;
  title?: string;
  children: React.ReactNode;
}
```
