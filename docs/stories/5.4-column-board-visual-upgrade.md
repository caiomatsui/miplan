# Story 5.4: Column & Board Visual Upgrade

## Story Info
- **Epic:** [Epic 5 - Professional UI/UX Polish](./epic-5-professional-ui-polish.md)
- **Points:** 5
- **Priority:** P1 (Container Polish)
- **Dependencies:** Story 5.1, Story 5.3

## User Story

**As a** user viewing my kanban board,
**I want** columns and the board to look polished and professional,
**So that** the interface feels like a premium productivity tool.

## Background

Current state:
- Columns have basic container styling
- Column headers are functional but plain
- Board background is flat
- Drop zones lack visual feedback
- Task counts are basic text

Target state (reference: Asana, Linear):
- Columns have subtle depth and separation
- Headers have count badges and optional color accents
- Board has subtle texture or gradient
- Drop feedback is clear and delightful
- Visual flow guides the eye

## Acceptance Criteria

### AC1: Column Container Polish
- [ ] Subtle background differentiation from board
- [ ] Rounded corners (12px) with subtle border
- [ ] Optional: Very subtle shadow for depth
- [ ] Consistent padding (12px internal)
- [ ] Smooth transition on drag-over state

### AC2: Column Header Redesign
- [ ] Title: 13px, semibold, proper truncation
- [ ] Task count: Badge pill style (e.g., "12" in muted pill)
- [ ] Drag handle: Visible on hover, subtle dots pattern
- [ ] Menu button: Appears on hover, proper alignment
- [ ] Optional status color accent (subtle top border or dot)

### AC3: Board Background
- [ ] Subtle gradient or texture (not flat color)
- [ ] Light mode: Very subtle warm/cool gradient
- [ ] Dark mode: Subtle depth gradient
- [ ] Should not distract from content
- [ ] Performance: Use CSS gradients, not images

### AC4: Drop Zone Feedback
- [ ] Column drop zone: Subtle highlight when dragging over
- [ ] Empty column: Dashed border placeholder
- [ ] Between tasks: Line indicator for insertion point
- [ ] Column reorder: Ghost outline of column position
- [ ] Colors use primary accent subtly

### AC5: Add Column Button
- [ ] Styled as dashed outline card
- [ ] Hover: Solid border, slight background
- [ ] Icon + text: "+ Add column"
- [ ] Same height as column header initially
- [ ] Positioned after last column

### Board Background
```css
.board-background {
  background: linear-gradient(
    135deg,
    oklch(var(--background)) 0%,
    oklch(var(--background) / 0.98) 50%,
    oklch(var(--muted) / 0.3) 100%
  );
}

.dark .board-background {
  background: linear-gradient(
    135deg,
    oklch(var(--background)) 0%,
    oklch(0.11 0.015 280) 100%
  );
}
```

### Count Badge Component
```tsx
function TaskCountBadge({ count }: { count: number }) {
  return (
    <span className="inline-flex items-center justify-center min-w-[1.25rem] h-5 px-1.5 text-xs font-medium bg-muted text-muted-foreground rounded-full">
      {count}
    </span>
  );
}
```
