# Miplan - Wireframes & UI Specifications

## Design Philosophy

| Principle | Implementation |
|-----------|----------------|
| **Anti-clutter** | Maximum white space, actions hidden until hover |
| **Clear hierarchy** | Narrow sidebar, board dominates the screen |
| **Instant feedback** | Visual states for drag, active timer |
| **Touch-friendly** | Minimum touch targets 44px |
| **Performance-first** | Minimal visual elements, no animations that block interaction |

```css
/* Background */
--bg-primary:      #FFFFFF;    /* Main background */
--bg-secondary:    #F9FAFB;    /* Column background */
--bg-tertiary:     #F3F4F6;    /* Hover states */

/* Text */
--text-primary:    #111827;    /* Headings, titles */
--text-secondary:  #6B7280;    /* Body text */
--text-muted:      #9CA3AF;    /* Placeholders, hints */
--text-inverse:    #FFFFFF;    /* On dark backgrounds */

/* Accent */
--accent:          #3B82F6;    /* Primary actions, links */
--accent-hover:    #2563EB;    /* Hover state */
--accent-light:    #DBEAFE;    /* Light accent bg */
```

## Responsive Breakpoints

| Breakpoint | Width | Layout Changes |
|------------|-------|----------------|
| **Mobile** | < 768px | Single column view, overlay sidebar, stacked tabs |
| **Tablet** | 768-1024px | Collapsed sidebar (icons), 2-3 columns visible |
| **Desktop** | 1024-1440px | Expanded sidebar, all columns visible |
| **Wide** | > 1440px | Wider columns, more spacing |

---

## Accessibility Checklist

| Requirement | Implementation |
|-------------|----------------|
| **Color contrast** | All text meets WCAG AA (4.5:1 min) |
| **Focus indicators** | Visible focus ring on all interactive elements |
| **Keyboard navigation** | Tab order follows visual order, shortcuts documented |
| **Screen readers** | ARIA labels on icons, live regions for updates |
| **Touch targets** | Minimum 44x44px on mobile |
| **Reduced motion** | Respect prefers-reduced-motion |
